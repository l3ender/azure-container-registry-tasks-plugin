After Azure Container Registry finishes building image, it pushes the image to registry with tags provided. Single Built image can be pushed with multiple repositories and tags.

Environment variables may be used in tag names - they will be replaced at build time. Examples:

  • repository:${ENV_VARIABLE}
  • repository:$ENV_VARIABLE
  • $ENV_VARIABLE:$ENV_VARIABLE
  • $ENV_VARIABLE

If no image name provided, Azure Container Registry **WILL NOT** push any image to registry after building.
